# React + TypeScript + Vite

## Структура проекта

- **main catalog**: Клиентская часть на React.
- **backend**: Серверная часть на Express.js.


## Установка и запуск

### Frontend:
- npm install
- npm run dev

### Backend:
- npm install
- npm start
